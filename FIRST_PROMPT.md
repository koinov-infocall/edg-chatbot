3. Създай src/db/schema.sql с таблиците: qa_pairs, chat_sessions, chat_messages, unmatched_questions, bot_config. Използвай точно колоните описани в CLAUDE.md.

5. Създай src/db/pool.js - PostgreSQL connection pool с DATABASE_URL от .env

6. Създай src/middleware/auth.js - JWT verification. Декодира token, извлича user_id, role, institution_id, institution_name и слага в req.user. 401 при невалиден token.

7. Създай src/services/llm.js - Claude Haiku API wrapper с @anthropic-ai/sdk. Model: claude-haiku-4-5, max_tokens: 500, temperature: 0.3. Включи prompt caching.

8. Създай src/services/matcher.js - matchQuestion(question, userRole, institutionId). Зарежда Q&A от DB по роля, конструира system prompt, извиква LLM, парсва за UNMATCHED маркер.

9. Създай src/services/context.js - buildSystemPrompt(userRole, institutionName, qaPairs). Използва шаблона от CLAUDE.md.

10. Създай src/routes/chat.js - POST /api/chat endpoint с пълния flow.

11. Създай src/routes/admin.js - CRUD endpoints за Q&A и логове.

12. Създай src/server.js - Express app с middleware (cors, helmet, rate-limit, auth), routes, error handling.

13. Създай .env.example и package.json scripts (dev, start, test, seed).

14. Създай public/widget.js - самостоятелен chat widget (vanilla JS, inline CSS, floating button долу-вдясно, чат прозорец, typing indicator).

Бъди изчерпателен. Искам работещ проект, който мога да стартирам с npm run dev.
